Manual de Usuario 📖

Ingresa los parámetros de tu problema de programación lineal a continuación. Selecciona el tipo de problema (maximizar o minimizar) y luego proporciona los coeficientes, restricciones y límites.

1. Definición del Problema


Define si quieres encontrar el valor más alto o más bajo de tu función objetivo.


Ingresa los coeficientes de las variables de tu función objetivo, separados por espacios (Ej: para 3x1 + 5x2, escribe "3 5").

2. Restricciones del Modelo

Define las limitaciones del problema. Asegúrate de que el número de coeficientes en cada restricción coincida con el número de variables de tu función objetivo.


Ingresa los coeficientes de cada restricción. Una restricción por línea, coeficientes separados por espacios (Ej: para 1x1 + 2x2, escribe "1 2").


Ingresa el tipo de relación para cada restricción. Una por línea. Puedes usar `<=`, `>=` o `=`. (Ej: para menor o igual, escribe "<=").


Ingresa el valor numérico del lado derecho para cada restricción. Uno por línea (Ej: "10").

Haz clic para calcular la solución óptima de tu problema de programación lineal.
